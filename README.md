# RetroStock
Users are "dropped" into a random historical date (e.g., January 2008) with $10,000. ​Key Mechanic: They "play" through historical years at an accelerated pace (1 minute = 1 month of real time). ​Goal: See if they would have survived the Great Recession or the Dot-com bubble. It’s an educational tool disguised as a high-stakes survival game.
